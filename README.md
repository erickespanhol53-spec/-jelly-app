# -jelly-app
The World of Jelly – MVP de App de Bem-Estar e Gamificação Protótipo funcional de um ecossistema focado em economia circular e saúde mental. O projeto utiliza gamificação e recompensas financeiras para estimular a desconexão (Modo Foco) e atividades saudáveis. Desenvolvido em HTML/JS para validação rápida de interface e fluxo de usuário.
